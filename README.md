# Wealth-Management

Wealth-Management is a Django-based web application focused on wealth and portfolio management.  
It is currently under active development and will eventually include features like client management, portfolio tracking, and reporting.

---

## Tech Stack

- **Language:** Python  
- **Framework:** Django  
- **Version Control:** Git & GitHub  
- **Environment Management:** `venv` (Python virtual environment)

---

## Repository

GitHub Repository:
https://github.com/narinderdev/Wealth-Management.git


## Clone the repository:
git clone https://github.com/narinderdev/Wealth-Management.git

## Prerequisites
Make sure you have installed:

- Python 3.10+
- Git
- A code editor like VS Code / PyCharm

## Create & activate a virtual environment
- Windows:
    python -m venv venv
    venv\Scripts\activate

- macOS / Linux:
    python3 -m venv venv
    source venv/bin/activate

## Install dependencies
    pip install -r requirements.txt

## Run the development server
    From the project root (where manage.py is located):
    - python manage.py runserver
